# علیرضا — توسعه‌دهنده نرم‌افزار و دانشجوی مهندسی کامپیوتر

## 👨‍💻 درباره من
سلام! من علیرضا هستم، دانشجوی مهندسی کامپیوتر و علاقه‌مند به توسعه نرم‌افزار، سیستم‌های هوش مصنوعی و طراحی معماری نرم‌افزار.
